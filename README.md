Python Cybersecurity Projects
Created by Michael Tran

This repository showcases Python scripts that demonstrate my journey toward becoming a cybersecurity engineer. These projects focus on key cybersecurity areas like password security, network scanning, and vulnerability assessment.

Projects
1. IP Geolocation Finder
Lookup IP address geolocation using the ipinfo.io API.

2. Password Strength Checker
Analyzes password strength and checks for common vulnerabilities.

3. Password Generator
Generates secure, random passwords based on user preferences.

4. Port Scanner
Scans a range of ports on a target host to identify open ports.

5. Web Vulnerability Scanner
Scans a website for common vulnerabilities like SQL injection and XSS.
